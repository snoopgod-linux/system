## Install `Apache2`

```shell
sudo apt install apache2
```

## Install `Chrome`

```shell
wget -O "/tmp/google-chrome.deb" "https://dl.google.com/linux/direct/google-chrome-stable_current_amd64.deb"
sudo dpkg -i /tmp/google-chrome.deb
```

## Install `Qt6`

```shell
sudo apt install qt6-base-dev qtcreator
```
